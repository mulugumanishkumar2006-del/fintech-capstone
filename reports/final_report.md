# 📊 Mutual Fund Analytics Dashboard

A comprehensive Mutual Fund Analytics Dashboard built using **Tableau Public** to analyze Indian mutual fund datasets. This project provides insights into fund performance, NAV trends, portfolio holdings, investor behavior, SIP analysis, benchmark indices, and industry statistics through interactive visualizations.

---

## 📌 Project Overview

The objective of this project is to transform raw mutual fund datasets into meaningful dashboards that help users understand market trends, fund performance, investor activity, and portfolio analytics.

The dashboards were developed as part of an analytics internship project using Tableau.
