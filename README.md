# 📌 Project Overview

The **Mutual Fund Analytics Platform** is an end-to-end Financial Data Analytics project that transforms raw mutual fund datasets into meaningful business insights through data engineering, SQL analysis, and interactive Tableau dashboards.

The project demonstrates the complete analytics lifecycle:

* Data Collection
* Data Cleaning
* Data Preprocessing
* Database Design
* SQL Analysis
* Exploratory Data Analysis (EDA)
* Interactive Dashboard Development
* Data Visualization
* GitHub Project Documentation

The primary objective is to help investors, analysts, and financial professionals understand mutual fund performance, industry trends, asset allocation, SIP growth, and fund house performance.

---

# 🎯 Project Objectives

* Analyze Indian Mutual Fund datasets.
* Build an end-to-end analytics workflow.
* Design a structured SQLite database.
* Perform data cleaning using Python.
* Develop interactive Tableau dashboards.
* Generate meaningful insights for investment analysis.

# 🚀 How to Run

### Clone Repository

```bash
git clone https://github.com/mulugumanishkumar2006-del/fintech-capstone.git
```

### Navigate to Project

```bash
cd fintech-capstone
```

### Install Dependencies

```bash
pip install pandas numpy matplotlib jupyter sqlite3
```

### Launch Jupyter Notebook

```bash
jupyter notebook
```

### Open Tableau Dashboard

Open the Tableau workbook (.twbx) using Tableau Public or Tableau Desktop.
